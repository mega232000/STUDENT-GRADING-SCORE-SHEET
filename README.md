# STUDENT-GRADING-SCORE-SHEET
This is one of the assignment  from data  starters community

## Project Overview
The **Student Grading Score Sheet** project is a data analysis and data cleaning project designed to organize and analyze students' academic scores using Microsoft Excel.

This project demonstrates how raw student score data can be structured, cleaned, and analyzed to produce meaningful academic results such as total scores, averages, and grades.

The goal of this project is to show practical skills in **data cleaning, spreadsheet analysis, and academic performance evaluation**.

---

## Problem Statement
Educational institutions often store student scores in raw formats that are difficult to analyze. This project provides a simple method for organizing student scores and computing academic performance metrics efficiently.

---

## Dataset
The dataset contains information about student scores in different subjects.

Typical fields include:

- Student Name
- Subject Scores
- Total Score
- Average Score
- Grade

---

## Tools Used
- Microsoft Excel
- Data Cleaning Techniques
- Spreadsheet Functions

Key Excel functions used include:

- SUM()
- AVERAGE()
- IF()
- Sorting and Filtering

---

## Project Workflow

1. Import raw student score data
2. Clean and organize the dataset
3. Calculate total scores
4. Compute student averages
5. Assign grades based on performance
6. Present results in a structured score sheet

---

## Key Features
- Organized student score dataset
- Automatic calculation of total scores
- Automatic grading system
- Easy-to-read score sheet
